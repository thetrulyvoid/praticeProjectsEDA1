# 📚 Data Structures I – Practical Projects (C)

This repository contains a collection of simple and practical projects developed for the **Data Structures I** course. The main goal is to reinforce fundamental concepts through hands-on implementation using the C programming language.

## 📌 Overview

The projects included here focus on core data structure concepts, such as:

- Arrays
- Linked Lists
- Stacks
- Queues
- Recursion
- Basic Algorithms (searching, sorting, etc.)

Each project is designed to be straightforward, educational, and easy to understand, serving as a learning resource rather than production-ready code.

## 🛠️ Technologies

- Language: **C**
- Compiler: GCC (recommended)
- Platform: Cross-platform (Linux, Windows, macOS)

Each folder represents a separate exercise or project related to a specific topic.

## ⚙️ How to Compile and Run

To compile the C programs, you can use **GCC**. Navigate to the project folder and run:

```bash
gcc main.c -o program
```

Then execute:

```bash
./program
```

### 🧩 Notes

* Some projects may contain multiple `.c` and `.h` files. In that case, compile all source files together:

```bash
gcc *.c -o program
```

* On Windows (using MinGW or similar), run:

```bash
program.exe
```

## 📖 Learning Purpose

These projects were created with the intention of:

* Practicing fundamental data structures
* Understanding memory management in C
* Improving problem-solving skills
* Preparing for more advanced computer science topics
